Law Search

Type a legal citation into the box below, and I'll try to send you to whatever it references:


Law Search recognizes vaguely Bluebook-style citations to a couple dozen sources of law, most notably every reporter in the Caselaw Access Project, and the whole U.S. Code as published by Cornell.

For case law, citations should follow VOLUME REPORTER PAGE[, PINCITE] format. For instance, "338 F.2d 708, 715" will go to page 715 of In re Kinsman Transit Co.. U.S. Code citations should look like TITLE usc SECTION[SUBSECTION]; "19 U.S.C. § 158(b)(4)" will go the secondary boycott ban and, for convenience's sake, so will "19 usc 158 b 4".

Other kinds of citation mostly match the Bluebook and state style guides. Most sources of law don't support subsections. For more info, see Sources of Law.

How it Works

Law Search is not, strictly speaking, a search engine. It does not list search results; instead it uses a set of rules to translate your query into a single URL, then takes you directly to that URL.

It does this by trying to match your query against a series of regular expressions (regexes), where each regex recognizes a single source of law. These sources of law are listed below. There are currently many unsupported sources; most notably the vast majority of state law.

Some of the regexes are very flexible; for instance, "42 usc 1988 b" is functionally identical to "42 U. S. Code § 1988(b)". Other sources are not as flexible. Court reporter names are relatively strict, because they must match the Caselaw Access Project (though capitalization is ignored). Likewise, the regexes are fairly strict about named chapters of state law. For these, I try to support the long form (e.g. "New York Rural Electric Cooperatives Law") and the common abbreviation (e.g. "NY REL"), but I make no promises. Not to mention, only a few states' laws are even recognized at the moment.

In any case, if your search query matches a regex, the program will generate a URL redirect that tries to take you directly to the relevant law on an external website. This may be a broken link, for instance if you enter a valid-looking citation to a code provision that doesn't exist.

One last note: From a technical standpoint, Law Search runs entirely in your browser, without any server-side logic. If you want to, you can even download this page and run the search locally without even connecting to my website; you'll just miss out on updates.
